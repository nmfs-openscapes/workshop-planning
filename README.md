# workshop-planning
The is a repo for calendaring and planning workshops using our workshop JupyterHub: https://workshop.nmfs-openscapes.2i2c.cloud/

This hub has the same features as our long-term hub, but it is designed for short-term events where you need to give participants access easily during a time-limited event (hours to days). 1 week after the event, the hub directories (and any participants) files are removed and the access code is changed.

To reserve the hub for your event, add an issue
